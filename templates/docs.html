users can use the endpoints below to fetch stock tickers from the S&P500
users can also filter by sector, currently there are 11 sectors in the S&P 500.

#python code example

import requests

url = "https://stock-tickers-api.herokuapp.com/sectors"
resp = requests.get(url)

print(resp,"\n")
print(resp.content)
